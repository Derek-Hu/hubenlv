### How to start
```
npm install -g yarn
yarn install
npm start
```

### Start mock server
```
npm run mock
```

### Analyzing the Bundle Size
```
npm run analyze
```

### How to test
```
yarn install
npm run test
```

### How to build
```
yarn install
npm run build
```

运行之后会生成`build`目录，其中包含所有的静态资源
